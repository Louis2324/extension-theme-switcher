# Extension Theme Switcher

A lightweight browser extension that lets you toggle between light and dark themes on any website.  

Part of the [Web Extension Pack](https://github.com/Louis2324/web-extension-pack), a collection of browser extensions built for learning, experimentation, and publishing.

---

## Features
- Toggle dark/light mode on any site with a single click
- Saves your preference per website
- Works on Chrome (Manifest V3)
- Minimal and fast

---

## Installation
1. Clone or download this repository
2. Open Chrome → go to `chrome://extensions/`
3. Enable **Developer mode** (top-right)
4. Click **Load unpacked** → select this repository folder
5. The extension icon will appear in the toolbar

---

## Usage
1. Click the extension icon in your browser toolbar
2. Toggle the theme for the current website
3. Your preference is saved automatically
